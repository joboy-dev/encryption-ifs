IPFS RELATED INSTALLATIONS
- Download IPFS using:
    - Get the version you want to use. I got go-ipfs 0.5.0 from https://github.com/ipfs/ipfs-desktop/releases so it can work with python ipfshttpclient package
    - I downloaded it as a tar.gz file from github
    - Extracted the file with `tar xzvf <file_name>` and ran the install.sh script
- Run:
    ipfs init
- Run:
    ipfs daemon

Install go
# Update your package list
sudo apt update

# Install Go
sudo apt install golang-go -y

# Verify the installation
go version

Set go path
Sometimes the system installs Go but doesn't add it to your profile. Add these lines to the end of your ~/.bashrc file:
export GOPATH=$HOME/go
export PATH=$PATH:/usr/local/go/bin:$GOPATH/bin
